[Tmux](https://github.com/tmux/tmux)
```sh
stow -vSt ~ tmux
```

#### Tmux Plugin Manager:
[Tpm](https://github.com/tmux-plugins/tpm)

```sh
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
Hit `prefix + I` to fetch the plugin and source it.
